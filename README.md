# 🌱 AgroNest

AgroNest is a smart agriculture web application built to help farmers manage crops, diseases, and predictions.

## 🚀 Features
- 🔐 Login / Signup UI
- 📊 Dashboard
- 👨‍🌾 Farmer Management (MongoDB)
- 🌾 Crops Information
- 🦠 Plant Diseases
- 🧪 Pesticides
- 🤖 ML-based Disease Prediction

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- MongoDB
- Render & Netlify

## 💡 Future Improvements
- Real AI Model
- Charts & Analytics
- Mobile App
